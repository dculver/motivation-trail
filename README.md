# The Motivation Trail

An educational game that teaches motivation research in educational psychology through an Oregon Trail-style adventure.

## About

**The Motivation Trail** is a pixel-art browser game designed for doctoral students in educational psychology. Players navigate through the challenges of graduate school while learning to apply real motivation theories from researchers like:

- Bandura (Self-Efficacy Theory)
- Deci & Ryan (Self-Determination Theory)
- Weiner (Attribution Theory)
- Dweck (Growth Mindset)
- Wigfield & Eccles (Expectancy-Value Theory)
- And many more!

## How to Play

1. Visit the game page
2. Click "BEGIN YOUR JOURNEY"
3. Read each scenario carefully
4. Choose responses based on motivation theory
5. Learn from the outcomes!
